## 🗄️ 네트워크 장비, NIC 및 케이블 분류표

 네트워크 통신은 데이터를 전송하는 **물리적 매체(케이블)**, 통신 능력을 부여하는 **인터페이스(NIC)**, 그리고 트래픽을 처리하고 경로를 결정하는 **장비(Device)**의 세 가지 핵심 요소로 구성됩니다.

다음은 이 요소들을 OSI 7계층 모델에 따라 분류하고 주요 역할을 요약한 표입니다.

---

### 1. 🔍 네트워크 장비/구성 요소

| 구성 요소 | OSI 계층 | 데이터 단위 | 주요 역할 |
| :---: | :---: | :---: | :--- |
| **케이블 (UTP, 광섬유)** | 1계층 (물리) | 비트 (Bit) | 데이터의 **물리적 전송 경로** 제공 |
| **NIC** (네트워크 인터페이스 카드) | 1계층/2계층 | 비트 / 프레임 | **PC에 통신 능력 부여**, MAC 주소 보유, 신호 변환 |
| **리피터, 허브** | 1계층 (물리) | 비트 (Bit) | 신호 증폭/재생, 단순 연결 |
| **스위치 (Switch)** | 2계층 (데이터 링크) | 프레임 (Frame) | **MAC 주소** 기반 통신, 충돌 도메인 분리 |
| **라우터 (Router)** | 3계층 (네트워크) | 패킷 (Packet) | **IP 주소** 기반 경로 결정, 네트워크 간 연결 |
| **L3 스위치** | 3계층 (네트워크) | 패킷 (Packet) | 라우터 기능 + 스위치 속도, 대규모 내부 라우팅 |
| **방화벽, 게이트웨이** | 3~7계층 | 패킷/데이터 | 보안 정책 적용, 프로토콜 변환 및 접근 제어 |
| **로드 밸런서** | 4~7계층 | 세그먼트/세션 | 트래픽 분산, 서버 가용성 및 성능 유지 |

---

### 2. 📝 구성 요소별 상세 설명

#### 🔸 1계층 요소: 물리적 전송 및 인터페이스

| 구성 요소 | 설명 |
| :--- | :--- |
| **케이블 (UTP/광섬유)** | **데이터가 이동하는 통로**입니다. UTP(Unshielded Twisted Pair)는 구리선을 사용하며 LAN에서 가장 일반적입니다. **광섬유 케이블**은 빛 신호를 사용하여 장거리 및 고속 전송이 필요할 때 사용됩니다. |
| **NIC (Network Interface Card)** | 컴퓨터에 고유한 **MAC 주소**를 부여하고, 컴퓨터의 디지털 데이터를 케이블을 통과할 수 있는 **전기 또는 광 신호**로 변환하는 장치입니다. |
| **허브/리피터** | **리피터**는 약해진 신호를 증폭합니다. **허브**는 연결된 모든 장치에 신호를 브로드캐스팅하는 단순 연결 장치입니다. |

#### 🔹 2계층 장비: 로컬 통신 및 분할

| 장비 | 설명 |
| :--- | :--- |
| **스위치 (Switch)** | **MAC 주소**를 학습하여 데이터를 원하는 목적지 포트로만 전송합니다. 브로드캐스팅을 제한하고 각 포트를 **독립적인 충돌 도메인**으로 만들어 네트워크 성능을 크게 향상시킵니다. |

#### 🔸 3계층 장비: 경로 결정 및 네트워크 간 연결

| 장비 | 설명 |
| :--- | :--- |
| **라우터 (Router)** | **IP 주소**를 기반으로 패킷이 서로 다른 네트워크를 넘어 목적지까지 도달할 수 있는 **최적의 경로**를 계산하고 전달합니다. **브로드캐스트 도메인**을 분리하여 광범위한 통신을 가능하게 합니다. |
| **L3 스위치** | 고속으로 L3 라우팅을 수행하는 장비로, 대규모 네트워크에서 빠른 속도의 내부 라우팅을 위해 사용됩니다. |

#### 🔹 4~7계층 장비: 보안 및 서비스 관리

| 장비 | 설명 |
| :--- | :--- |
| **방화벽 (Firewall)** | 사전에 정의된 규칙(IP, 포트, 프로토콜)에 따라 트래픽을 검사하고 차단하여 내부 네트워크를 보호하는 보안 장비입니다. |
| **로드 밸런서** | 서비스의 부하를 여러 서버에 효율적으로 나누어 주어, 서비스의 중단 없이 안정적인 운영을 가능하게 합니다. |

---

다음으로, LAN 환경에서 가장 많이 사용되는 **UTP 케이블의 규격(Cat.5e, Cat.6 등)과 그 차이점**에 대해 자세히 살펴보시겠습니까?
