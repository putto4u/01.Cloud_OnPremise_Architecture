## Packet Tracer 설치 및 환경 구성

네트워크 입문자와 전문가 모두에게 필수적인 시뮬레이션 도구인 **Cisco Packet Tracer**를 설치하는 과정은 단순히 파일을 내려받는 것 이상의 의미가 있습니다. 최신 버전은 Cisco Networking Academy 계정과의 연동이 필수적이므로, 올바른 절차를 따르는 것이 중요합니다.

---

### 1. Packet Tracer 다운로드 및 계정 생성

Packet Tracer는 Cisco에서 무료로 제공하지만, 공식 다운로드 경로를 이용하기 위해서는 **Skills for All** 또는 **Networking Academy** 계정이 필요합니다.

* **공식 다운로드 페이지:** [Cisco Networking Academy](https://www.netacad.com/portal/resources/packet-tracer)
* **권장 절차:**
1. [Skills for All](https://skillsforall.com/) 사이트에 접속하여 회원가입을 진행합니다 (Google 계정 연동 가능).
2. 'Getting Started with Cisco Packet Tracer' 코스에 등록합니다 (무료).
3. 코스 내부의 **Download** 링크를 통해 운영체제(Windows, Linux, macOS)에 맞는 설치 파일을 내려받습니다.



> **[실전 팁]**
> 과거에는 계정 없이 'Guest Mode'를 사용할 수 있었으나, 최신 버전(8.x 이상)은 로그인하지 않을 경우 파일 저장 횟수가 제한되거나 기능 사용이 불가능할 수 있습니다. 반드시 계정을 생성하고 로그인 상태를 유지하세요.

---

### 2. 운영체제별 설치 요구 사양

안정적인 시뮬레이션 환경을 위해 아래의 최소 사양을 확인하십시오. 복잡한 토폴로지를 구성할수록 CPU와 RAM의 자원 소모가 커집니다.

| 항목 | 최소 사양 | 권장 사양 |
| --- | --- | --- |
| **CPU** | Intel Pentium 4, 2.53 GHz 이상 | Intel Core i3 3.0 GHz 이상 |
| **RAM** | 2 GB | 4 GB 이상 |
| **저장공간** | 1.4 GB 이상의 여유 공간 | SSD 권장 |
| **OS** | Windows 10/11 (64bit) | 최신 업데이트 완료된 OS |

---

### 3. 설치 가이드 (Windows 기준)

1. **설치 파일 실행:** 다운로드한 `.exe` 파일을 실행합니다.
2. **라이선스 동의:** `I accept the agreement`를 선택하고 Next를 클릭합니다.
3. **경로 선택:** 기본 설치 경로를 유지하는 것을 권장합니다.
4. **추가 작업:** 바탕화면 바로가기 생성 여부를 선택합니다.
5. **설치 완료:** `Install` 버튼을 눌러 과정을 마칩니다.
6. **로그인 인증:** 프로그램 실행 시 나타나는 로그인 창에서 생성한 **Skills for All** 계정으로 로그인합니다.

---

### 4. 자주 발생하는 오류 및 해결책 (Troubleshooting)

* **로그인 루프 현상:** 로그인을 했음에도 계속 로그인 창이 반복된다면, 기본 브라우저의 쿠키와 캐시를 삭제하거나 브라우저를 크롬(Chrome)으로 설정한 뒤 재시도하십시오.
* **방화벽 차단:** 시뮬레이션 중 패킷 전송이 원활하지 않다면 Windows 방화벽에서 Packet Tracer의 통신 허용 여부를 확인해야 합니다.
* **글꼴 크기 문제:** 고해상도(4K) 모니터 사용 시 글자가 너무 작게 보인다면 `Options` -> `Preferences` -> `Font` 탭에서 크기를 조절할 수 있습니다.

---

### 5. Packet Tracer 활용의 핵심

Packet Tracer는 단순히 선을 연결하는 도구가 아닙니다. 실제 장비의 **CLI(Command Line Interface)**를 그대로 재현하므로, 아래의 모드를 적극 활용해야 합니다.

* **Realtime Mode:** 실제 네트워크 장비처럼 동작하는 실시간 모드.
* **Simulation Mode:** 패킷이 이동하는 경로와 OSI 7계층 데이터를 단계별로 캡처하여 분석할 수 있는 모드. (학습 시 가장 중요)
